How to build your firmware (no GitHub, no PlatformIO setup)
  1. Click Download firmware builder — you get a single angleros-build.ps1 with your settings baked in.
  2. Right-click it → Run with PowerShell (or run powershell -ExecutionPolicy Bypass -File .\angleros-build.ps1).
  3. It installs PlatformIO if needed, builds Marlin, and drops firmware.uf2 on your Desktop.
  4. Hold BOOT on the SKR Pico, plug in USB, and copy firmware.uf2 onto the RPI-RP2 drive.

Requires Git and Python on your PC. Windows may block a downloaded script — right-click → Properties → Unblock, or use the -ExecutionPolicy Bypass command above.
