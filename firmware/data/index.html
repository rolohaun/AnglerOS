How to build your firmware (no GitHub, no PlatformIO setup)
  1. Click your OS's Build firmware button. You get a single script (angleros-build.ps1 or .sh) with your settings baked in.
  2. Windows: right-click and Run with PowerShell (or powershell -ExecutionPolicy Bypass -File .\angleros-build.ps1).
    macOS/Linux: run bash angleros-build.sh.
  3. It installs PlatformIO if needed, builds Marlin, and drops the firmware on your Desktop.
  4. Flash it the way your board expects. The script prints exact steps for BOOTSEL/UF2 or SD card flashing.

Requires Git and Python. Windows may block a downloaded script. Right-click, Properties, Unblock, or use the -ExecutionPolicy Bypass command above.
